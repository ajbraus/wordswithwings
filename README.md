wordswithwings
==============

####How to run the app:

#####Install node.js:
http://nodejs.org/download/

#####Install bower:
npm install bower -g

#####Clone Project: 

1. git clone https://github.com/ajbraus/wordswithwings.git
2. cd wordswithwings
3. npm install
4. bower install
5. npm start
6. In your browser goto url: http://localhost:3000
